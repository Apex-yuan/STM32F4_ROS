## 注意事项：

【1】 在以固定频率发布数据时，一定不要忽略**“=”**

例如：

```c
//错误代码
if(t - tTime[0] > 1000/200)
{
    publishImuMsg();
    tTime[0] = t;
}
//正确代码
if(t - tTime[0] >= 1000/200)
{
    publishImuMsg();
    tTime[0] = t;
}
```

对比上述代码，只有一个等号的差别，但在实际测试中消息的发布频率差距很大（错误代码的发布频率约为160hz左右，而正确代码的发布频率可以达到要求的200hz）。具体原因还不清楚，后续如果清楚了原因也要记录到此。
